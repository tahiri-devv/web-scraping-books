# 📚 Web Scraping Books Project

Un script Python de web scraping qui extrait automatiquement les livres depuis le site **Books to Scrape** et parcourt plusieurs pages.

---

## 🚀 Fonctionnalités

- Scraping automatique des pages du site
- Extraction des informations des livres :
  - 📖 Titre du livre
  - 💰 Prix
- Pagination automatique (navigation entre les pages)
- Limitation du nombre de pages pour test (modifiable)
- Affichage clair dans le terminal

---

## 🛠️ Technologies utilisées

- Python 🐍
- Requests 🌐
- BeautifulSoup (bs4) 🍲
- urllib
